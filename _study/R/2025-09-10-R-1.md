# R

- Useful functions

  ```r
  ls() # 현재 사용중인 객체 이름 확인
  args() # 함수의 인자 구조 확인
  ```

- Basic Operations

  ```r
  die <- 1:6
  die %*% die # matrix multiplication (행렬곱)
  die %o% die # vector outer product (벡터 외적)
  round(3.1415) # 반올림
  factorial(3) # !
  ```

- 비복원 추출 / 복원 추출

  ```r
  die <- 1:6
  sample(die, size = 2) # 비복원 추출
  dice <- sample(die, size = 2, replace = TRUE) # 복원 추출
  dice 
  dice # 여러번 호출해도 값은 동일
  ```
